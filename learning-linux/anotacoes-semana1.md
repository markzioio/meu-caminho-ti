# Semana 1 - Linux Básicos (20-26 de Julho)

## O que aprendi

### Dia 1-2: Terminal e Navegação
- `ls` — lista arquivos
- `cd` — muda de pasta
- `pwd` — mostra pasta atual
- `mkdir` — cria pasta nova
- `rm` — deleta arquivo
- `rmdir` — deleta pasta vazia

### Dia 3-4: Permissões
- `chmod 755 arquivo` — muda permissões
- `chown usuario arquivo` — muda dono
- `ls -l` — mostra detalhes
